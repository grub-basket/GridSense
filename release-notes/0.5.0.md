## GridSense 0.5.0

- Empty draft rows at the top and bottom of every grid. Drafts are UI-only: no note is created until the row has a name and you press Enter — then the note is created, any property values you typed flush into it, and a fresh draft appears. Abandoning or clearing a draft never touches your files, and clearing cells in a real row never deletes the note.
- Excel-style row stability: once displayed, row order is frozen — editing cells (even the sorted column) and newly created notes never reshuffle the grid. Changing sort, filter, or pressing ↺ re-derives the order. Notes committed from a draft stay pinned to the end of the grid where you made them.
- Fixed: a background recompile could drop its follow-up refresh, leaving the grid on stale rows.
