## GridSense 0.4.0

- Inline frontmatter takeover: Obsidian's native properties panel is replaced by GridSense's keyboard-friendly editor in Live Preview and Reading mode (toggle in settings; on by default). Arrow keys navigate, Tab switches key/value, Enter edits, Cmd/Ctrl+Enter adds, Cmd/Ctrl+Backspace deletes, Cmd/Ctrl+Z undoes.
- The properties modal and inline editor now share one component.
- README: install from the community plugin store.
