## GridSense 0.4.1

- Inline properties takeover is now OFF by default while in beta (Settings toggle).
- Property rows have a type button on the left: change property type (Text/List/Number/Checkbox/Date/Date & time), zoom the value, or delete — reserved properties (tags, aliases, cssclasses) are recognized and locked to their list types.
- Values parse by property type: tags strip #, lists split on commas or newlines, numbers and checkboxes coerce properly.
- List values render as chips with a visible x per item (removal is undoable).
- Left/right arrows switch between key and value (same as Tab).
- Fixed: double-clicking a value no longer kicks you out of edit mode; Tab after editing stays in the properties strip instead of jumping to the note body.
- New: cell zoom — a large editor for one value, in the property menu and the grid's right-click menu.
