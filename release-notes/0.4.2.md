## GridSense 0.4.2

- Removed the silent default row cap (was 2,000): a fresh grid now always shows every note in scope, so you never mistake a truncated grid for the whole folder. Virtualized rendering keeps big folders fast.
- Prominent row counter in the toolbar: "1,234 rows", or "60 of 1,234 rows" (highlighted) whenever a filter or row limit is trimming the view — with a tooltip saying which.
- Grids over 10,000 rows show a warning button that opens the columns & views modal to hide columns or set a row limit.
